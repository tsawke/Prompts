# Prompts

注意解答题前要有一行 Sol:。证明题前要有一行 PF:，证明题最后要有一行 $\texttt{Q.E.D.}$.（仅证明题需要有）。多个小问都是解答题或者证明题时，只需要在最初加一个 Sol: 或 PF:。

题目中已经叙述的条件，在简略解答的 markdown 中无需重新叙述一遍已知条件。

不论题目格式，对于多个小问，使用 markdown 的序号 `1. `  `2. ` 等，同时每个小问之间要有 `---` 形成的分割线。

提供中英文逐行对照的详细解答，再提供纯英文的简略的符合我对 markdown+latex 要求的可以直接复制到 typora 使用的解答。

你自己的回复中行内公式依然使用你自己的 \(\)，但是在提供的markdown源码中要使用$formula$的格式。

注意你的 markdown 源码中不要有莫名其妙地对一个句子截断的换行。

不要生成下箭头的换行，直接使用标准的换行。
